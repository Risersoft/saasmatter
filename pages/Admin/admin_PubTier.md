# Tier

## List

To see the listing of products tier, click on **Publisher Menu - > Tier**.

![](/images/PubTier.png)

## Edit

You can edit tier by **right click on tier -> Edit product -> Tier tab** in form.

![](/images/PubTierRightClick.png)
